What is Computer Vision?

Computer Vision is a field of artificial intelligence that enables machines to interpret and understand visual information from the world. It involves developing algorithms and systems that can automatically perform tasks that the human visual system can do.

Key Insight: Computer vision systems can analyze images and videos to identify objects, faces, scenes, and activities, enabling applications from autonomous vehicles to medical diagnosis.

How Computers See Images

Digital Image Representation

Computers represent images as multi-dimensional arrays of pixel values. Each pixel contains color information, typically represented in RGB (Red, Green, Blue) format.

Image Dimensions:

  • Height: Number of rows of pixels
  • Width: Number of columns of pixels
  • Channels: Color depth (RGB = 3 channels)
# Image as numpy array
import numpy as np
from PIL import Image

# Load an image
img = Image.open('example.jpg')
img_array = np.array(img)

print(f"Image shape: {img_array.shape}")
# Output: (height, width, channels) = (224, 224, 3)

# Access pixel values
pixel = img_array[100, 150]  # RGB values [R, G, B]
print(f"Pixel at (100, 150): {pixel}")
# Output: [245, 134, 67]

Core Computer Vision Tasks

Image Classification

Assigning a single label to an entire image from a predefined set of categories.

Example: Is this image a cat, dog, or car?

Object Detection

Locating and classifying multiple objects within an image, drawing bounding boxes around them.

Example: Find all cars and pedestrians in a street scene

Semantic Segmentation

Classifying every pixel in an image, creating a pixel-level understanding of the scene.

Example: Which pixels belong to road, buildings, sky?

Face Recognition

Identifying and verifying individuals based on facial features in images or videos.

Example: Unlock phone with face, find person in photos

Convolutional Neural Networks (CNNs)

CNN Architecture

CNNs are specialized neural networks designed for processing grid-like data such as images. They use convolution operations to detect local features and pooling for dimensionality reduction.

Key Components:

  • Convolutional Layers: Apply filters to detect features
  • Pooling Layers: Reduce spatial dimensions
  • Activation Functions: Introduce non-linearity
  • Fully Connected Layers: Final classification

Feature Detection Process

Layer 1: Low-level Features

Edges, corners, color gradients

Layer 2: Mid-level Features

Shapes, textures, patterns

Layer 3: High-level Features

Object parts, semantic concepts

Final Layer: Classification

Object categories, scene understanding

# Simple CNN Implementation
import torch
import torch.nn as nn
import torch.nn.functional as F

class SimpleCNN(nn.Module):
    def __init__(self, num_classes=10):
        super(SimpleCNN, self).__init__()
        # First convolutional block
        self.conv1 = nn.Conv2d(3, 32, kernel_size=3, padding=1)
        self.bn1 = nn.BatchNorm2d(32)
        self.pool1 = nn.MaxPool2d(2, 2)
        
        # Second convolutional block
        self.conv2 = nn.Conv2d(32, 64, kernel_size=3, padding=1)
        self.bn2 = nn.BatchNorm2d(64)
        self.pool2 = nn.MaxPool2d(2, 2)
        
        # Third convolutional block
        self.conv3 = nn.Conv2d(64, 128, kernel_size=3, padding=1)
        self.bn3 = nn.BatchNorm2d(128)
        self.pool3 = nn.MaxPool2d(2, 2)
        
        # Fully connected layers
        self.fc1 = nn.Linear(128 * 4 * 4, 512)
        self.dropout = nn.Dropout(0.5)
        self.fc2 = nn.Linear(512, num_classes)
        
    def forward(self, x):
        # First block
        x = self.pool1(F.relu(self.bn1(self.conv1(x))))
        
        # Second block
        x = self.pool2(F.relu(self.bn2(self.conv2(x))))
        
        # Third block
        x = self.pool3(F.relu(self.bn3(self.conv3(x))))
        
        # Flatten for fully connected layers
        x = x.view(-1, 128 * 4 * 4)
        
        # Fully connected layers
        x = F.relu(self.fc1(x))
        x = self.dropout(x)
        x = self.fc2(x)
        
        return x

# Image Preprocessing and Augmentation
import torchvision.transforms as transforms
from PIL import Image

# Define preprocessing pipeline
transform = transforms.Compose([
    transforms.Resize((224, 224)),  # Resize to 224x224
    transforms.RandomHorizontalFlip(p=0.5),  # Random horizontal flip
    transforms.RandomRotation(degrees=15),  # Random rotation ±15 degrees
    transforms.ColorJitter(brightness=0.2, contrast=0.2, saturation=0.2),  # Color adjustments
    transforms.ToTensor(),  # Convert to tensor
    transforms.Normalize(mean=[0.485, 0.456, 0.406],  # ImageNet normalization
                         std=[0.229, 0.224, 0.225])
])

# Apply transformations
image = Image.open('example.jpg')
transformed_image = transform(image)

print(f"Transformed image shape: {transformed_image.shape}")
# Output: torch.Size([3, 224, 224])

Evaluation Metrics

Classification Metrics

Accuracy

Overall correctness of predictions

Accuracy = Correct Predictions / Total Predictions

Precision

Accuracy of positive predictions

Precision = TP / (TP + FP)

Recall

Coverage of actual positive cases

Recall = TP / (TP + FN)

Object Detection Metrics

IoU (Intersection over Union)

Overlap between predicted and ground truth bounding boxes

IoU = Area of Overlap / Area of Union

mAP (mean Average Precision)

Average precision across all classes and IoU thresholds

mAP = Average of AP values

F1-Score

Harmonic mean of precision and recall

F1 = 2 × (Precision × Recall) / (Precision + Recall)
