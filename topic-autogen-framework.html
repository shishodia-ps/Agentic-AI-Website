What is AutoGen?

AutoGen is a revolutionary framework developed by Microsoft Research that enables the creation of next-generation applications using multiple collaborating agents. It provides a flexible and powerful infrastructure for building conversational agents that can work together to solve complex tasks through automated chat.

Key Insight: AutoGen simplifies the orchestration, automation, and optimization of complex LLM workflows by enabling multi-agent conversations that can automatically solve tasks through collaborative intelligence.

AutoGen Architecture

AutoGen enables multiple specialized agents to collaborate through automated conversations

Core AutoGen Concepts

Conversable Agents

Agents that can participate in conversations, send and receive messages, and perform actions based on received messages.

  • • AssistantAgent - AI-powered assistants
  • • UserProxyAgent - Human-in-the-loop agents
  • • Custom agent creation
  • • Agent specialization

Automated Chat

Dynamic conversation patterns where agents automatically communicate to solve tasks through collaborative dialogue.

  • • Two-agent conversations
  • • Group chat orchestration
  • • Dynamic conversation flow
  • • Termination conditions

Code Execution

Built-in code execution capabilities that allow agents to write, test, and execute code as part of problem-solving.

  • • Python code execution
  • • Docker containerization
  • • Safe execution environments
  • • Result validation

Nested Chats

Hierarchical conversation structures where complex problems are broken down into smaller, manageable sub-conversations.

  • • Problem decomposition
  • • Sub-task coordination
  • • Result aggregation
  • • Context management

Key Features & Benefits

Multi-Agent Orchestration

Seamlessly coordinate multiple agents with different roles and capabilities to solve complex problems.

  • • Role-based agent specialization
  • • Dynamic agent creation
  • • Conversation management
  • • Task decomposition

Code Generation & Execution

Agents can write, execute, and debug code automatically as part of their problem-solving process.

  • • Automatic code generation
  • • Safe execution environments
  • • Error handling and debugging
  • • Result validation

Human-in-the-Loop

Integrate human feedback and oversight at any point in the conversation flow for enhanced control.

  • • Interactive feedback

  • • Context injection
  • • Override capabilities

Implementation Examples

Basic Two-Agent Conversation 

# Basic AutoGen Two-Agent Setup
import autogen

# Configuration for OpenAI API
config_list = [
    {
        'model': 'gpt-4',
        'api_key': 'your-openai-api-key'
    }
]

# Create Assistant Agent
assistant = autogen.AssistantAgent(
    name="assistant",
    llm_config={
        "config_list": config_list,
        "temperature": 0.7
    },
    system_message="""You are a helpful AI assistant.
You provide clear and accurate information.
You can help with coding, analysis, and problem-solving."""
)

# Create User Proxy Agent
user_proxy = autogen.UserProxyAgent(
    name="user_proxy",
    human_input_mode="NEVER",  # No human input required
    max_consecutive_auto_reply=10,
    is_termination_msg=lambda x: x.get("content", "").rstrip().endswith("TERMINATE"),
    code_execution_config={
        "work_dir": "coding",
        "use_docker": False
    }
)

# Start the conversation
user_proxy.initiate_chat(
    assistant,
    message="Write a Python function to calculate the factorial of a number and test it with 5!"
)

# The conversation will automatically continue until termination

Multi-Agent Group Chat 

# Multi-Agent Group Chat with Specialized Roles
import autogen

# Configuration
config_list = [{'model': 'gpt-4', 'api_key': 'your-api-key'}]

# Create specialized agents
planner = autogen.AssistantAgent(
    name="planner",
    llm_config={"config_list": config_list},
    system_message="""You are a strategic planner. Break down complex problems into manageable steps.
Create detailed plans and identify key milestones."""
)

coder = autogen.AssistantAgent(
    name="coder",
    llm_config={"config_list": config_list},
    system_message="""You are an expert Python developer. Write clean, efficient, and well-documented code.
Focus on best practices and error handling."""
)

critic = autogen.AssistantAgent(
    name="critic",
    llm_config={"config_list": config_list},
    system_message="""You are a code reviewer. Analyze code for bugs, performance issues, and improvements.
Provide constructive feedback and suggestions."""
)

# User proxy for human oversight
user_proxy = autogen.UserProxyAgent(
    name="user_proxy",
    human_input_mode="TERMINATE",  # Ask for human input before terminating
    max_consecutive_auto_reply=10,
    is_termination_msg=lambda x: x.get("content", "").find("TERMINATE") >= 0,
    code_execution_config={"work_dir": "project", "use_docker": False}
)

# Create group chat
agents = [user_proxy, planner, coder, critic]
groupchat = autogen.GroupChat(
    agents=agents,
    messages=[],
    max_round=20
)

# Create group chat manager
manager = autogen.GroupChatManager(
    groupchat=groupchat,
    llm_config={"config_list": config_list}
)

# Start the collaborative task
user_proxy.initiate_chat(
    manager,
    message="""Create a web scraping tool that extracts product information from an e-commerce site.
The tool should:
1. Handle pagination
2. Extract product name, price, and description
3. Save data to CSV
4. Include error handling and rate limiting

Let me know when you're ready to start."""
)

# Agents will automatically collaborate to complete the task

Agent with Custom Tools 

# Custom Agent with Tool Integration
import autogen
from typing import Dict, Any

# Custom tool functions
def web_search(query: str) -> str:
    """Search the web for information."""
    # Implementation would use a search API
    return f"Search results for: {query}"

def calculate(expression: str) -> str:
    """Evaluate mathematical expressions."""
    try:
        result = eval(expression)
        return str(result)
    except:
        return "Error: Invalid expression"

# Configuration
config_list = [{'model': 'gpt-4', 'api_key': 'your-api-key'}]

# Create assistant with function calling capabilities
assistant = autogen.AssistantAgent(
    name="assistant",
    llm_config={
        "config_list": config_list,
        "functions": [
            {
                "name": "web_search",
                "description": "Search the web for current information",
                "parameters": {
                    "type": "object",
                    "properties": {
                        "query": {"type": "string", "description": "Search query"}
                    },
                    "required": ["query"]
                }
            },
            {
                "name": "calculate",
                "description": "Perform mathematical calculations",
                "parameters": {
                    "type": "object",
                    "properties": {
                        "expression": {"type": "string", "description": "Mathematical expression"}
                    },
                    "required": ["expression"]
                }
            }
        ]
    },
    system_message="""You are a helpful assistant with access to web search and calculator tools.
Use these tools when needed to provide accurate and current information.
Always explain your reasoning and show your work."""
)

# User proxy that can execute functions
user_proxy = autogen.UserProxyAgent(
    name="user_proxy",
    human_input_mode="NEVER",
    max_consecutive_auto_reply=10,
    function_map={
        "web_search": web_search,
        "calculate": calculate
    }
)

# Register the functions
user_proxy.register_function(
    function_map={
        "web_search": web_search,
        "calculate": calculate
    }
)

# Start conversation with tool usage
user_proxy.initiate_chat(
    assistant,
    message="What's the current population of Tokyo and what percentage of Japan's total population does that represent? Use the search tool to find current data and the calculator to compute the percentage."
)

Nested Conversations 

# Nested Chat for Complex Problem Solving
import autogen

# Configuration
config_list = [{'model': 'gpt-4', 'api_key': 'your-api-key'}]

# Create agents for main conversation
main_assistant = autogen.AssistantAgent(
    name="main_assistant",
    llm_config={"config_list": config_list},
    system_message="""You are the main coordinator. Break down complex problems into sub-tasks
and delegate them to specialized agents. Synthesize results and provide final solutions."""
)

user_proxy = autogen.UserProxyAgent(
    name="user_proxy",
    human_input_mode="NEVER",
    max_consecutive_auto_reply=10
)

# Create specialized sub-agents for nested conversations
def create_sub_conversation(task_type: str, task_description: str):
    """Create a nested conversation with specialized agents."""
    
    if task_type == "data_analysis":
        analyst = autogen.AssistantAgent(
            name="data_analyst",
            llm_config={"config_list": config_list},
            system_message="You are a data analyst expert. Analyze data and provide insights."
        )
        
        visualizer = autogen.AssistantAgent(
            name="visualizer",
            llm_config={"config_list": config_list},
            system_message="You are a data visualization expert. Create charts and graphs."
        )
        
        # Create nested chat
        nested_agents = [analyst, visualizer]
        nested_groupchat = autogen.GroupChat(
            agents=nested_agents,
            messages=[],
            max_round=5
        )
        
        nested_manager = autogen.GroupChatManager(
            groupchat=nested_groupchat,
            llm_config={"config_list": config_list}
        )
        
        # Start nested conversation
        analyst.initiate_chat(
            nested_manager,
            message=f"Analyze the following dataset and create visualizations: {task_description}"
        )
        
        return nested_groupchat.messages

# Main conversation that uses nested chats
main_assistant.register_function(
    function_map={
        "create_sub_conversation": create_sub_conversation
    }
)

# Start main conversation
user_proxy.initiate_chat(
    main_assistant,
    message="""I need to analyze customer data for our e-commerce business.
The dataset contains purchase history, customer demographics, and product information.
Please:
1. Perform exploratory data analysis
2. Create visualizations showing trends
3. Provide insights on customer behavior
4. Suggest improvements for our business"""
)
